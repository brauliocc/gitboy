# GitBoy - Habit Contribution Graph

A habit contribution graph styled like a classic Game Boy. Track your daily contributions with a nostalgic retro gaming aesthetic.

## Overview

GitBoy is a simple habit tracker that displays your contributions in a GitHub-style contribution graph, but with the iconic green-on-green color scheme of the original Game Boy.

## Features

- Interactive contribution grid showing a full year (364 days)
- Click any cell to add a contribution
- Press the "A" button to add a contribution for today
- Press the "B" button to reset all contributions
- Contributions are saved to local storage
- Shows total contributions and current streak

The application runs on port 5000 using Node.js and Express:

```bash
node server.js
```
